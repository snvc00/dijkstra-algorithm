Cargar y crear un grafo construido a partir de una imagen es un proceso simple.

Es necesario dar clic en el botón "Cargar", seleccionar la imagen deseada en la ventana emergente y finalmente crear un grafo que represente dicha imagen utilizando el botón "Crear Grafo".

Ejecutar el Algoritmo de Dijkstra en dos pasos.

A pesar de que este algoritmo solo requiere un grafo y un vértice inicial para ser ejecutado, en la implementación también se requiere seleccionar un destino para que se pueda crear una animación que recorra este camino en caso de su existencia. En la aplicación se mostraran listas desplegables con las opciones para cada vértice, se requiere dar clic en el botón "Siguiente" para aplicar los cambios.

Seleccionar una Partícula para Animación

Después de seleccionar el vértice destino, aparecerán en pantalla tres imagenes que podrán ser utilizadas como partículas móviles en una animación que muestre el camino encontrado, se puede utilizar cualquiera, para seleccionarla solo se necesita arrastrarla al grafo.

Movimiento de un lugar a otro.

Una vez seleccionada la partícula, se realizara la animación que señale el camino mínimo encontrado, ahora el último vértice destino será el vértice inicial y se podrá elegir un nuevo destino, seleccionandolo en la lista desplegable y dando clic en el botón "Siguiente" para guardar, ejecutando el mismo procedimiento que el recorrido anterior. Este comportamiento puede ser ejecutado de manera sucesiva, además de que la partícula seleccionada puede ser modificada.

Capacidad de Reiniciar y Continuar con el mismo grafo.

El botón "Reiniciar" tiene la función de revertir todos los cambios realizados al momento hasta el punto en donde se acababa de crear un grafo a partir de la imagen seleccionada, este procedimiento no puede ser revertido, pero se puede volver a iniciar desde la selección de un vértice origen.
